# CutoutLab

Website skeleton for an `image background remover` product site built with Next.js.

## Scripts

```bash
npm install
npm run dev
```

## Current state

- Responsive marketing homepage tailored to the background remover use case
- Upload interaction with client-side validation and preview
- Placeholder API route at `src/app/api/remove-background/route.ts`
- Product requirement document in `PRD.md`

## Next step
